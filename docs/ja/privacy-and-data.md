# プライバシーとデータの扱い（概要）

このページは概要です。**正式かつ最新のデータの取り扱いは、プロダクトの
[プライバシーポリシー](https://usecaseify.com/privacy) に従います。**
本ページとポリシーに差異がある場合、ポリシーが優先します。

> **デプロイ済みポリシーとの最終照合：** 2026年7月13日

## プロダクトの挙動として確認できること

- **リサーチは公開 Web ページのみ**を対象とし、クロール範囲には上限が
  あります。非公開データベースは使用しません。
- あなたが入力した製品情報・アップロード資料は、プロダクトの機能
  （プロダクトプロファイル作成、リサーチ、生成）を提供するために処理
  されます。
- プロダクトユーザーが能動的に行った採否、修正指示、品質評価は、評価時点の
  内容・モデル・prompt version とともに品質問題の分析用に保存します。Settings
  で許可した場合、UseCaseify 専用モデルの評価・改善・学習にも利用できます。
  無効化すると、まだ学習用に書き出されていない既存記録と今後の記録を
  学習対象外にします。
- アカウント・決済情報、および Validation 回答者の連絡先・回答は学習対象に
  しません。非公開データを第三者の汎用 AI モデル学習目的で提供しません。
- **検証ページの回答者はアカウント不要・匿名で回答**でき、同意は明示的に
  取得・記録されます。連絡先の記入は任意で、回答データと分離して管理
  されます。
- レポートと検証ページの共有リンクは、いつでも失効させられます。

## 現在利用するサービス区分

デプロイ済みプロダクトは現在、AI 生成・評価に OpenAI API、Web 検索に
Serper、文書解析に MinerU（設定時）、認証・データ保存に Supabase、決済に
Stripe、利用状況の統計分析に Google Analytics を利用します。各操作に必要な
範囲のデータだけを送信します。委託先の詳細および今後の変更については、
この概要ではなく [プライバシーポリシー](https://usecaseify.com/privacy) が
正となります。

## 詳細はプライバシーポリシーへ

データの保存期間、第三者サービスの利用、削除の取り扱いなどの詳細は
[プライバシーポリシー](https://usecaseify.com/privacy) をご確認ください。
ご不明点は [support@usecaseify.com](mailto:support@usecaseify.com) まで。
