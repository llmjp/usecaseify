# UseCaseify 日本語ドキュメント

**市場が反応する use case を、顧客事例がまだない段階で見つける。**

UseCaseify（[usecaseify.com](https://usecaseify.com/)）は、出典をたどれる
市場エビデンスと実在の見込み顧客からのフィードバックをもとに、「誰に・どの
ユースケースを・どんな価値提案で」市場に出すかを決められる、ユースケース
発見・検証プラットフォームです。このサイトは、UseCaseify が現在何であり、
どう動き、何を主張しないかを示す公式リファレンスです。

- [プロダクトファクト]({{ site.baseurl }}/ja/product-facts/) —
  検証可能な現在の定義・機能・制限・ステータス（正とするページ）
- [方法論]({{ site.baseurl }}/ja/methodology/) —
  市場エビデンス、AI の推論、合成プレチェック、実在の見込み顧客の
  フィードバックをどう区別しているか
- [ホワイトペーパー]({{ site.baseurl }}/ja/whitepaper/) —
  顧客事例がない段階の GTM 意思決定を構造化する方法（v1.0, 2026年7月）
- [エンドツーエンド例]({{ site.baseurl }}/ja/examples/end-to-end-example/) —
  架空の製品でワークフロー全体を通す例
- [ユースケース検証ガイド]({{ site.baseurl }}/ja/guides/how-to-validate-a-use-case/) —
  実践 8 ステップ
- [スコアリングフレームワーク]({{ site.baseurl }}/ja/guides/use-case-scoring-framework/) —
  10 項目の意味
- [UseCaseify と ChatGPT の違い]({{ site.baseurl }}/ja/comparisons/usecaseify-vs-chatgpt/)
- [よくある質問]({{ site.baseurl }}/ja/faq/) — 現行プロダクトに即した Q&A
- [用語集]({{ site.baseurl }}/ja/glossary/) — UseCaseify が使う用語の公式定義
- [AI の限界]({{ site.baseurl }}/ja/ai-limitations/) —
  AI に分かること・分からないこと
- [プライバシーとデータの扱い]({{ site.baseurl }}/ja/privacy-and-data/) —
  プロダクトのプライバシーポリシーを正とする概要
