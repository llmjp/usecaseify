# UseCaseify 日本語ドキュメント

UseCaseify（[usecaseify.com](https://usecaseify.com/)）は、顧客事例がまだ
十分にそろっていないプロダクトチームのための、ユースケース発見・検証
プラットフォームです。このサイトは、UseCaseify が現在何であり、どう動き、
何を主張しないかを示す公式リファレンスです。

- [プロダクトファクト]({{ site.baseurl }}/ja/product-facts/) —
  検証可能な現在の定義・機能・制限・ステータス
- [方法論]({{ site.baseurl }}/ja/methodology/) —
  市場エビデンス、AI の推論、合成プレチェック、実在の見込み顧客の
  フィードバックをどう区別しているか
- [よくある質問]({{ site.baseurl }}/ja/faq/) — 現行プロダクトに即した Q&A
- [用語集]({{ site.baseurl }}/ja/glossary/) — UseCaseify が使う用語の公式定義
- [ホワイトペーパー]({{ site.baseurl }}/ja/whitepaper/) —
  顧客事例がない段階の GTM 意思決定を構造化する方法（v1.0, 2026年7月）

English documentation is available at
[English pages]({{ site.baseurl }}/en/).
