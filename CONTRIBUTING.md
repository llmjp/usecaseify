# Contributing

This repository holds the official public documentation for UseCaseify. It is
maintained by the UseCaseify team at LLM JAPAN, LLC. It does not contain product
source code, and UseCaseify itself is not an open-source project, so code
contributions are not applicable here.

## What contributions are useful?

- Corrections of factual errors in any documentation page
- Reports of broken links or rendering problems
- Fixes for unnatural wording or translation mistakes in the English or
  Japanese pages
- Questions about unclear statements (please open an issue)

## What is out of scope?

- Marketing or SEO-oriented content additions
- Fictional case studies, invented customers, or unverified product claims
- Product feature requests and support questions — please contact
  [support@usecaseify.com](mailto:support@usecaseify.com) instead
- Changes that state planned or future capabilities as if they were current

## Source requirements

Any change to a factual statement must be verifiable against the current
product at [usecaseify.com](https://usecaseify.com/) or an official
announcement. Statements that cannot be verified will not be merged.

## Pull requests

- Keep pull requests small and focused on one correction
- Write the description in English or Japanese
- Maintainers review all changes; there is no guarantee that an external pull
  request will be accepted
