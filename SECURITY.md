# Security Policy

This repository contains documentation only; the UseCaseify product source
code is not hosted here.

## Reporting a vulnerability

If you believe you have found a security vulnerability in the UseCaseify
product ([usecaseify.com](https://usecaseify.com/)) or a problem with this
documentation site, please email
[support@usecaseify.com](mailto:support@usecaseify.com) with the details.
Please do not open a public issue for security reports.

We will acknowledge reports as quickly as we can. Please give us reasonable
time to investigate and address an issue before any public disclosure.
